# Caso Práctico: Impacto del calendario competitivo en el rendimiento deportivo

## Descripción del Proyecto

Este proyecto analiza cómo el calendario competitivo (frecuencia de partidos, descansos, viajes y tipo de competición) impacta en el rendimiento de un equipo de fútbol profesional. El objetivo es mejorar la toma de decisiones deportivas mediante el estudio de variables como días de descanso, ubicación de los partidos, y rendimiento a lo largo de la temporada.

## Estructura del Proyecto

- **`app.py`**: Aplicación principal de Streamlit
- **`data/`**: Datos crudos y procesados
- **`etl_scripts/`**: Scripts de limpieza y transformación de datos
- **`utils/`**: Módulos de análisis y visualización
- **`docs/`**: Documentación e informes del proyecto
- **`pages/`**: Páginas adicionales de la aplicación
- **`reports/`**: Reportes generados
